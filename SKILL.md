# Daily Financial News - 每日财经新闻搜罗

## 触发条件

当用户提及以下内容时使用此技能：
- "今天有什么新闻"
- "今天的财经新闻"
- "国内外重大新闻"
- "金融市场发生了什么"
- "有什么影响市场的消息"

也可用于定时任务/心跳检查，定期推送新闻摘要。

## 核心工作流

### 1. 搜索国际新闻

**搜索范围：**
- 国际地缘政治事件（冲突、制裁、会议）
- 主要央行政策（美联储、欧央行、日央行等）
- 大宗商品价格波动（黄金、石油、金属）
- 重大公司财报（FAANG、英伟达、特斯拉等）
- 全球宏观经济数据（GDP、CPI、就业）

**搜索策略：**
```
搜索关键词（根据当天日期调整）：
- "today financial news impact markets"
- "breaking news economy today"
- "Fed policy interest rate news"
- "stock market today earnings"
- "geopolitics oil gold price"
```

### 2. 搜索国内新闻

**搜索范围：**
- 国务院/部委重大政策
- 宏观调控政策（货币、财政、产业）
- 行业监管动态
- 重大改革措施
- 重要会议/讲话

**搜索关键词：**
```
- "中国 重大政策 新闻 今天"
- "国务院 发布会 今日"
- "央行 外汇 政策 最新"
- "国内 财经新闻 今天"
```

### 3. 筛选与分类

**高优先级（直接影响市场）：**
- 地缘政治冲突 → 金价、油价、避险资产
- 央行利率决策 → 汇率、债券、股市
- 大公司财报（FAANG、英伟达等） → AI科技板块
- 经济数据超预期 → 相关货币/商品
- 重大制裁或贸易政策 → 供应链相关资产

**中优先级（间接影响）：**
- 行业趋势报告
- 重要人物讲话
- 政策征求意见稿

**低优先级（一般新闻）：**
- 常规会议
- 中小公司财报
- 地方性新闻

### 4. 输出格式

```markdown
## 📈 今日财经新闻摘要 [YYYY-MM-DD]

### 🌎 国际重点

**地缘政治/宏观**
- [新闻标题]
  - 来源：[媒体]
  - 影响：金价/油价/股市/汇率...
  - 简评：...

**公司财报**
- [公司名] 财报要点
  - 影响：相关板块...
  - 简评：...

**央行动态**
- [央行] 最新政策
  - 影响：...
  - 简评：...

### 🇨🇳 国内重点

**政策动态**
- [政策名称]
  - 来源：官网/媒体
  - 影响：...
  - 简评：...

**行业监管**
- [行业] [政策要点]
  - 影响：...

---

### 💡 今日市场影响总结
[1-2句话总结今日关键影响因素]
```

## 注意事项

1. **当天新闻为主**：优先搜索24小时内的新闻
2. **权威来源**：优先引用路透社、彭博社、华尔街日报、新华社等权威媒体
3. **影响分析**：每条新闻简要说明对金融市场的潜在影响
4. **时间精力**：搜索和整理时间控制在合理范围内，优先处理高优先级新闻
5. **避免重复**：如果用户每天都需要，可以记录已发送的新闻避免重复

## 定时任务建议

如需每日自动推送，可配合cron设置：
- 时间：建议北京时间早上9点前（汇总隔夜国际新闻+当日国内新闻）
- 频率：每日一次
- 输出：推送到指定渠道
